# Obscuro
Obscuro is a real time messaging application, built with React, Socket.io, Expressjs, and MongoDB.

## Project Status 
This project is currently in development.

### Current Features
- Login / Sign Up / Logout / Session
- View messages
- Send message
- Read receipts
- Create conversation

## Future features
- Like messages
- Profile page
- Add friend
- Avatar pictures
- Avatar / Message border
- Number of Likes
- Guest User
